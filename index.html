Как выглядит обычный плагин для ядра EasyCore

Main.php

 namespace plugin;

 use pocketmine\plugin\PluginBase;

 class Main extends PluginBase{
	public function onEnable(){
		$this->getLogger()->info("Hello world!");
	}
 }

plugin.yml

 name: ExamplePlugin
 main: plugin\Main
 api: 2.0.0 //Обязательно используйте 2.0.0
 version: 1.0 //Версия плагина
 author: dixsin //Автор плагина

Что мы сделали в самой новой версии ядра?

 Список изменений для серверного ПО EasyCore версии 1.1.4:

 - Исправлены некоторые проблемы, связанные со стабильностью сервера
 - Исправлены ошибки, которые могли приводить к падению сервера при большой нагрузке
 - Добавлена поддержка новых команд, которые позволяют управлять игроками и настройками сервера 
 - Улучшена производительность и общая эффективность работы сервера
 - Исправлены некоторые ошибки в коде, которые могли вызывать неправильное поведение сервера

 Это основные изменения, которые были внесены в версию EasyCore 1.1.4.
